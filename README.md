# Langchain Sandbox To Rag

A collection of LangChain experiments and learning exercises.

## Topics Covered

- Prompt Templates
- LCEL Chains
- Output Parsers
- Pydantic Output Parsing
- Chat Memory
- Text Splitters
- Document Loaders
- Ollama Integration
- RAG Foundations

## Setup

```bash
git clone <repo-url>
cd langchain-labs

python -m venv myvenv

myvenv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
python src/lcel_chains.py
```
